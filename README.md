# 🚗 Vehicle Rental Management System

## 📌 Project Description
This is a full-stack Vehicle Rental Management System where users can book vehicles online and make payments.

---

## 🚀 Features
- User Registration & Login
- Vehicle Booking System
- Razorpay Payment Integration
- Admin Dashboard (Manage Vehicles & Bookings)
- GPS Vehicle Tracking
- Rating & Feedback System

---

## 🛠️ Tech Stack
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB
- Payment: Razorpay
